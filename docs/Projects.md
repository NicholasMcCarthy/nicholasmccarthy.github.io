# Projects

## Open Source and Engineering

- [AmpliGraph](https://ampligraph.org/) - Open source Python library for
  link prediction and representation learning on knowledge graphs.
- PathXL Algorithm Framework - Scalable plugin framework and utility suite for
  digital pathology image analysis, implemented in C++11 with OpenCV and
  OpenMP.
- [WekaLab](https://github.com/NicholasMcCarthy/wekalab) - Matlab library for
  working with the Weka machine learning framework.

## Applied Research Highlights

- Weakly supervised deep learning for ACL tear segmentation and classification
  with radiologist-level performance.
- Differential privacy approaches for network generation and knowledge graph
  protection.
- Knowledge graph and latent-space methods for molecule discovery and
  counterfactual explanations.
